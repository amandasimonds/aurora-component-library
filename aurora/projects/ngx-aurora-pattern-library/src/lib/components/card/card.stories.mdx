# Card Component

-   Card sizes can be defined by `auto`, `md`, or `sm`.
-   `auto` sized cards will expand to whatever width is needed for the content inside.
-   `sm` and `md` have fixed widths depending on what is needed.
-   The gap of 8px between items is already defined - however, a button should have a margin of 16px when inside a card, so you must add a `margin-top-xxs` and/or `margin-bottom-xxs` class to the button component when using a button inside a card.
